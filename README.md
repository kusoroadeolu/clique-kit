# clique-kit

A collection of CLI libraries and utilities built on [Clique](https://github.com/kusoroadeolu/Clique).

---

## Modules

| Artifact | Description                          |
|----------|--------------------------------------|
| `veneer` | A CLI syntax highlighting library    |
|  `cli-timer`| A simple and configurable CLI timer| 

---

## Usage

Each module is published independently to Maven Central. Pull only what you need.

### Maven

```xml
<dependency>
    <groupId>io.github.kusoroadeolu</groupId>
    <artifactId>veneer</artifactId>
    <version>1.1.0</version>
</dependency>
```

### Gradle

```groovy
implementation 'io.github.kusoroadeolu:veneer:1.0.0'
```

---

## Requirements

- Java 21+ for `veneer`
